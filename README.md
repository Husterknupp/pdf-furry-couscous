# pdf-furry-couscous

PDF magic for researching tasks

## Getting started

`npm install` to get the libraries that we need.

Install PDF libraries yourself:

-   mutool
-   pdftk
-   pdfjam

Find instructions via `npm start` (same as `node index.js --help`)
