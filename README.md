# pdf-furry-couscous

PDF magic for researching tasks

## Getting started

Install PDF libraries yourself:

-   mutool
-   pdftk
-   pdfjam

`npm install` to get the js libraries that we need

Find usage instructions via `npm start` (will call the script's `--help` section)
